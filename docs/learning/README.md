### 场景 6：磁盘空间不足/需要清理
**推荐文档**：
1. [镜像和缓存管理](./guide.md#镜像和缓存管理) - 完整的清理指南
2. [快速参考 - 缓存管理](./reference.md#镜像存储和缓存管理) - 快速清理命令

**快速操作**：
```bash
# 1. 查看磁盘使用
docker system df

# 2. 清理构建缓存（安全，释放 12+ GB）
docker buildx prune -a -f

# 3. 清理悬空镜像
docker image prune

# 4. 清理停止的容器
docker container prune
```

**重要提示**：
- ✅ 清理构建缓存**不会**影响已构建的镜像和运行中的容器
- ✅ GalOS 实测：清理缓存释放了 12.35 GB 空间，镜像功能完全正常
- ⚠️ 下次构建镜像时会变慢（需要重新执行所有步骤）
